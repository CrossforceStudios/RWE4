# Rodblogan Warfare Engine 4

This plugin serves as both the **engine and editor layer** for Rodblogan Warfare–inspired systems.

It is an open-source, modular shooter framework derived from the original Rodblogan Warfare game, designed to be extended beyond its original setting into other shooter genres.

This includes (but is not limited to):
- Tactical modern combat experiences
- Sci-fi shooter systems (e.g. custom universe projects)
- Cinematic or franchise-inspired gameplay prototypes (e.g. Star Wars-style demos)
- Fully custom shooter mechanics built on the same core architecture
---

## 🎯 Purpose

Rodblogan Warfare Engine 4 provides a reusable foundation of shooter systems and editing tools.

It is designed to allow developers to:
- Recreate the Rodblogan Warfare experience
- Modify or replace its systems freely
- Build entirely new shooter genres on top of the same framework
- Rapidly prototype combat-focused gameplay systems
---

## 🧠 Design Philosophy

While rooted in Rodblogan Warfare, this engine is intentionally **abstracted and modular**.

Core systems are built to be:
- Genre-agnostic (not limited to modern military combat)
- Replaceable and extensible
- Compatible with multiple gameplay styles and settings
- Suitable for both realistic and cinematic shooter experiences

---

## ⚙️ Key Idea

This plugin is not a standalone game.

It is a **tooling and runtime framework** built from Rodblogan Warfare’s original systems, designed to be reconfigured into new shooter experiences across different genres.
